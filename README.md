# Stackly Electronic Mobility

A modern, responsive website for an electric mobility company with stunning animations and user-friendly interfaces.

## 🚀 Features

### Pages Included
1. **Home Page** - Landing page with hero section, products overview, statistics, and testimonials
2. **Login Page** - User authentication with email and password
3. **Register Page** - New user registration with form validation
4. **About Page** - Company information, mission, vision, team, and milestones
5. **Blog Page** - Articles with search and category filtering
6. **Products Page** - Detailed product showcase with specifications and comparison table

### Design Features
- ⚡ **Modern Animations** - Smooth transitions and engaging animations throughout
- 📱 **Fully Responsive** - Works perfectly on desktop, tablet, and mobile devices
- 🎨 **Professional Color Scheme** - Modern cyan, dark blue, and white color palette
- 🔧 **Interactive Elements** - Hover effects, form validation, and dynamic content filtering
- 📊 **Statistics Dashboard** - Animated counter for impressive metrics
- 🎯 **User-Friendly Navigation** - Smooth navigation with dropdown menus

## 📝 Customization

### Change Colors
Edit the CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #00D4FF;
    --secondary-color: #0A3A4A;
    --accent-color: #FF6B6B;
    /* ... */
}
```

### Add Content
- Edit the HTML files directly
- Update text, images, and links
- Modify product specifications
- Add new blog articles

### Modify Animations
- Adjust animation durations in CSS
- Change animation delays
- Create new keyframe animations
- Customize hover effects
